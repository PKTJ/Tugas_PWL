Flow Program:
1. User mengakses index.php → index() menampilkan daftar.
2. Klik “Tambah” → add() menampilkan form.
3. Submit form → store() menambahkan data.
4. Klik “Edit” → edit(id) menampilkan form prefilled.
5. Submit form → update(id) mengubah data.
6. Klik “Hapus” → delete(id) menghapus data.
